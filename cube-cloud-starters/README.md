# 魔方微服务组件

### 项目结构说明

```
├─cube-starter 微服务拓展功能
│  ├─starter-cloud 微服务核心模块：单体升级为微服务
│  ├─starter-job 分布式任务调度
│  ├─starter-lock 分布式锁
│  ├─starter-rabbitmq 基于RabbitMQ的消息总线
│  ├─starter-rocketmq 基于RocketMQ的消息总线
│  ├─starter-redis 分布式redis
│  ├─starter-seata 分布式事务
│  ├─starter-shardingsphere 分库分表
```
